Description

Zip Install is a Pligg module that allows you to upload templates and modules to their respective Pligg CMS folders.

The module only uploads only ZIP archives. The uploaded archive then extracts the contents to a folder named after the archive file name.

For example file advance_js.zip create /advance_js directory and unzips all of the files to that folder.

Do not use directories in the archive reflecting the module directory (/modules/module_name/). The module or template files should be in the root directory of the archive file, not in a subfolder.
