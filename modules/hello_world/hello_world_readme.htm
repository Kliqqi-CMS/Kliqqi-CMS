Hello World Module

Created by: Eric "Yankidank" Heikkinen
Last Updated: 07/24/2012

Description

This module dispalys a series of hint messages to the admin user to introduce them to the different parts of the Admin Panel. It uses browser cookies to make sure that you only have to view each hint once, so be careful before dismissing messages or you'll have to clear your cookies to view it again.
